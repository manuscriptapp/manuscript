File format

Plain folders, plain files

A project is just a folder with Markdown files and JSON metadata. Open it in any editor, commit to Git, write scripts against it.

MyNovel.manuscript/ ├── project.json ├── contents/ │ ├── draft/ │ │ ├── 01-chapter-one.md │ │ └── 02-chapter-two.md │ └── notes/ └── snapshots/
