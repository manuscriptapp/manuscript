## Version 1.0 — Initial Release

Manuscript is a native writing app for novelists, screenwriters, and long-form writers who value privacy and file ownership.

### Write
- Rich text editor with formatting toolbar and Markdown persistence
- Composition (focus) mode for distraction-free writing
- Reading mode with centered A4-style layout
- Split editor to view two documents side by side
- Find and replace across your manuscript
- Comments and inline notes without cluttering your text
- Document snapshots to save and compare versions
- Writing targets with daily progress tracking

### Organize
- Binder sidebar with chapters, scenes, and folders
- Drag-and-drop reordering and cross-folder moves
- Corkboard view for visual scene planning
- Outliner view with sortable columns
- Character and location databases with custom fields
- World map with Look Around street-level preview
- Icon and color customization for documents and folders

### Import and Export
- Import from Scrivener projects (with icons, colors, and writing history)
- Import from Markdown, plain text, PDF, HTML, and Word (DOCX on macOS)
- Export to PDF, EPUB, HTML, Markdown, plain text, and Scrivener 3
- Compile multiple documents into a single output
- Attach images and PDFs directly to documents
- Native print on iOS and macOS

### AI Assistance
- On-device AI powered by Apple Foundation Models (private by default)
- Optional cloud AI with OpenAI and Anthropic Claude models
- Bring your own API key — no hidden data sharing

### Your Files, Your Way
- Markdown-based package format you own forever
- iCloud Drive sync across all your Apple devices
- Automatic backup management
- No required accounts, no subscriptions
- Open source under the MPL-2.0 license
